# Microsoft Copilotとは
Microsoftが提供する、生成AIサービスの総称です。

OpenAI社が提供する最新の言語モデルをベースとしており、様々な業務のアシスタントを行います。

様々なMicrosoft製品と統合し、チャットだけでなく、クラウドサービス・Office製品のサポートも行うことができます。


# Copilotの種類
Copilotはサービスの総称です。実際は、Microsoftのサービスごとにサポートする内容や利用条件が異なります。

| サービスの名称 | 内容 |
| --- | --- |
| Microsoft Copilot Chat | ブラウザやデスクトップアプリから利用できるチャットアプリ |
| Microsoft 365 Copilot | Microsoft 365 製品のアシスタントサービス |
| Copilot for Dynamics 365 |  Dynamics 365のアシスタントサービス |
| Microsoft Security Copilot | AIを利用したセキュリティ分析サービス |
| GitHub Copilot | VSCode上で利用できる、プログラミング支援サービス |
| Copilot Studio | 自分でAIエージェントをカスタマイズできるサービス |

> [!CAUTION]  
> 他にも種類があったり、今後名称変更が行われる場合があります。  
> 最新の情報と異なる点がありましたら、Issue等で報告してください。
